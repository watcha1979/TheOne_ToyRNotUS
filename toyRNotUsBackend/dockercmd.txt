Docker
=======
Start docker and run spring boot app
 docker container run -d -v $(pwd)/toystore.jar:/xxx/toystore.jar -p 8081:8080 openjdk:8-jre java -jar /xxx/toystore.jar 
-d = deamon
-v = put file to docker image
-p = port map
--name web = assign name

Build docker image
* prepare Dockerfile first
docker image build -t toystore:0.1 .
 
List all docker process 
docker container ps 

Run docker iamge
docker container run -p 8081:8080 toystore:0.1

Stop docker
docker container stop <docker name>


List all docker image
docker image

Docker image name convention when use docker hub
<usename>/toystore:0.1

Tag image locally
docker image tag toystore:0.1 somkiat/toystore:0.1

Push image to docker hub
docker image push somkiat/toystore:0.1

Pull image from docker hub for build
docker pull maven:3.5.2-alpine

Run maven on docker
docker container run -v $(pwd)/.:/xxx -w /xxx maven:3.5.2-alphine mvn clean package
